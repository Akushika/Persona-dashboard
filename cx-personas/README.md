## Structure

Each persona file includes:
- **Quote**: Persona's defining statement
- **Overview**: Day-to-day activities and responsibilities
- **Key Responsibilities**: Primary job functions
- **Challenges**: Pain points and obstacles
- **Responsibilities (JTBD)**: Jobs to be done
- **Product Needs**: Required features and capabilities
- **SaaS sections**: SaaS-specific content where applicable
